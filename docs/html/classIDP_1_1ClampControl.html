bool IDP::ClampControl::bobbin_present (  )

See if a bobbin is in the jaw.

Used especially when navigating down the rack to check when we've found something. The red and green bobbins block a lot of reflected light from the metal backplate, so use this large drop in light to notice them. The white bobbins reflect about the same amount, but when present without lights on they will block a good deal of light from hitting the sensor anyway, so we can detect them as such.

Returns:
True when a bobbin is found
